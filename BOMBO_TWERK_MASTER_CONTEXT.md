# BOMBO TWERK — MASTER CONTEXT

## Project Overview

Bombo Twerk is a premium mexican performancewear brand focused on:

* twerk
* pole dance
* nightlife-inspired fashion
* expressive feminine confidence
* movement culture
* social/TikTok-native fashion experiences

This is NOT a generic ecommerce store.
This is a fashion-performance digital brand.

The ecommerce experience should feel:

* aspirational
* premium
* editorial
* emotionally expressive
* mobile-first
* culturally alive
* movement-focused
* socially driven

# Language Rules

The public-facing ecommerce experience must be written in Spanish.

All:
- headlines
- CTAs
- product storytelling
- collection names
- navigation
- checkout labels
- marketing copy
- UI messaging

should default to Spanish.

Internal code, variables, APIs and technical implementation can remain in English.

The tone of Spanish copy should feel:
- emotionally expressive
- premium
- confident
- feminine
- nightlife-oriented
- editorial
- natural for mexican audiences

Avoid:
- literal english translations
- corporate ecommerce spanish
- generic latin-american marketing phrases
